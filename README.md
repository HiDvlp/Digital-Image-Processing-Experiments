# 数字图像处理实验
课程实验共包含7个实验内容，至少选做其中3个。
## 第1次实验-使用局部直方图统计的图像增强
本次实验主要依据《数字图像处理（第 4 版）》（冈萨雷斯著）中的相关理论，重点研究**直方图统计方法**在**图像增强**中的应用。实验将对比**全局直方图均衡化**与**局部处理**方法的差异，并通过**基于统计量的局部增强**算法，针对性地提升图像暗区域的细节可见度。

实验内容涵盖算法原理验证、参数影响分析以及针对特定图像特性的编程优化。通过查阅资料，学生需掌握不同的像素邻域遍历方法，分析其运行效率；并通过调整关键参数，观察图像增强效果的变化规律，最终实现对含隐藏细节图像的有效增强。

该实验代码的使用说明见`Experiments/chap_1/README.md`文件。
